= redmine_improved_searchbox

Converts Redmine[http://www.redmine.org]'s project jumpbox with searchable Select2[http://ivaynberg.github.com/select2] combobox.

= Requires

This is variant which is not embedding select2 library so it is not conflicting with different plugins.

- Select2 library from different plugin (i.e. rmnet a_common_libs).

= Features

- filtering in top redmine project list

= Compatible Versions

- Redmine 2.1.x - 3.4.x
